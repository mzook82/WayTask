# Changelog

All notable changes to this project will be documented in this file.

---

# [1.0.0] - In Progress

## Added

- Initial TestFlight beta release
- Shopping list management
- Interactive map
- Nearby store discovery
- Store recommendations
- Camera integration
- Photo capture
- Photo library import
- Tap to focus
- Pinch to zoom
- Flash / Torch support
- Local notifications
- Apple Maps navigation
- Store details
- Website integration
- User settings
- Category preferences

## Improved

- Modern shopping interface
- Interactive map experience
- Camera usability
- Navigation flow
- Dark theme support
- Standardized recommendation language across Home, Shopping, Map, and nearby notifications.
- Kept Home recommendations concise while preserving detailed product coverage inside Shopping.

## Known Issues

- Start Shopping button does not respond.
- Shopping Plan flow is incomplete.
- Store database may contain outdated locations.

## Fixed

- Active shopping sessions now resume automatically after the app is relaunched.

# [1.0.1] - Unreleased

## Fixed

- Completed the Shopping Flow.
- Added store selection before starting shopping.
- Added active Shopping Mode.
- Added shopping session persistence.
- Added automatic session recovery after application relaunch.
- Added Resume Shopping support.
- Prevented duplicate active shopping sessions.

## Improved

- Removed the Shopping Summary card.
- Redesigned the Shopping product row using a compact two-tier layout.
- Improved readability for long product names.
- Preserved all Shopping interactions while improving the layout.
- Replaced percentage-based store coverage with complete shopping-list coverage.
- Improved recommendation wording to better reflect estimated availability.
- Removed misleading "Best" terminology from shopping recommendations.
- Added "Likely here" and "Other items" product grouping.
- Added user-friendly estimation messaging.

## Known Limitations

- Store coverage is based on estimated product availability.
- Shopping Plan is recreated when needed after a cold launch.
