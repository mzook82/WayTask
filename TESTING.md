# WayTask Testing Checklist

This checklist must be completed before every TestFlight release.

---

# General

- [ ] App launches successfully
- [ ] No crashes during startup
- [ ] Dark mode works correctly
- [ ] Navigation between all tabs works
- [ ] No obvious UI issues

---

# Shopping

## Product List

- [ ] Add product
- [ ] Edit product
- [ ] Delete product
- [ ] Search products
- [ ] Product categories work correctly

## Shopping Flow

- [ ] Shopping Plan opens
- [ ] Store can be selected
- [ ] Start Shopping works
- [ ] Shopping session begins successfully

---

# Camera

- [ ] Camera opens
- [ ] Camera permission works
- [ ] Photo capture works
- [ ] Import from Photos works
- [ ] Tap to focus works
- [ ] Pinch to zoom works
- [ ] Flash / Torch works

---

# Map

- [ ] Current location is displayed
- [ ] Stores appear correctly
- [ ] Store details open
- [ ] Apple Maps navigation works
- [ ] Website link opens
- [ ] Nearby notifications work

---

# AI

- [ ] Gemini connection works
- [ ] AI response received
- [ ] Product recognition works (if enabled)

---

# Notifications

- [ ] Local notification received
- [ ] Notification opens the app correctly

---

# Settings

- [ ] Categories can be updated
- [ ] Preferences are saved

---

# Performance

- [ ] No significant UI lag
- [ ] No memory issues noticed
- [ ] No unexpected loading delays

# Beta Validation

- [ ] Critical issues fixed
- [ ] High-priority issues tested
- [ ] Regression testing completed
